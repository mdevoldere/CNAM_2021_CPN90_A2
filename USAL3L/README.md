# CNAM CPN90 A2 USAL3L

## Logiciels requis :

- Un serveur web (Apache ou NGinx)
- Un serveur de bases de données (MySQL 5.7+ ou MariaDB 10.3+)
- PHP 7.4+

Vous pouvez installer Laragon, Xampp ou Wamp pour disposer des 3 logiciels nécessaires.
